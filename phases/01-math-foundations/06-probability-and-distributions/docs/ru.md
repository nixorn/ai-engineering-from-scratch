## Проблема

Классификатор выдает `[0.03, 0.91, 0.06]`. Языковая модель выбирает следующее слово из 50 000 кандидатов. Диффузионная модель генерирует изображения, семплируя из обученных распределений. Все это - вероятность в действии.

Каждое предсказание модели - это вероятностное распределение. Каждая функция потерь измеряет, насколько далеко предсказанное распределение от истинного. Каждый шаг обучения подстраивает параметры, чтобы одно распределение выглядело как другое. Без вероятности вы не сможете прочитать ни одну статью про ML, отладить ни одну модель или понять, почему ваша training loss это NaN.

## Концепция

### События, пространства элементарных событий и вероятность

Пространство элементарных событий S - это множество всех возможных исходов. Событие - это подмножество пространства элементарных событий. Вероятность отображает события на числа от 0 до 1.

```
Бросок монеты:
  S = {Г, Р}
  P(Г) = 0.5,  P(Р) = 0.5

Бросок одной кубика:
  S = {1, 2, 3, 4, 5, 6}
  P(четное) = P({2, 4, 6}) = 3/6 = 0.5
```

Три аксиомы определяют всю теорию вероятностей:
1. P(A) >= 0 для любого события A
2. P(S) = 1 (что-нибудь всегда случается)
3. P(A или B) = P(A) + P(B), когда A и B не могут произойти одновременно

Все остальное (теорема Байеса, ожидания, распределения) следует из этих трех правил.

### Условная вероятность и независимость

P(A|B) - это вероятность события A при условии, что произошло B.

```
P(A|B) = P(A и B) / P(B)

Пример: колода карт
  P(Король | Фигура) = P(Король и Фигура) / P(Фигура)
                      = (4/52) / (12/52)
                      = 4/12 = 1/3
```

Два события независимы, когда знание одного ничего не говорит вам о другом:

```
Независимы:     P(A|B) = P(A)
Эквивалентно:   P(A и B) = P(A) * P(B)
```

Броски монеты независимы. Вытягивание карт из колоды без возврата - нет.

### Функция вероятности масс (PMF) vs Функция плотности вероятности (PDF)

Дискретные случайные величины имеют функцию вероятности масс (PMF). Каждый исход имеет определённую вероятность, которую можно прочитать напрямую.

```
PMF: P(X = k)

Честный кубик:
  P(X = 1) = 1/6
  P(X = 2) = 1/6
  ...
  P(X = 6) = 1/6

  Сумма всех вероятностей = 1
```

Непрерывные случайные величины имеют функцию плотности вероятности (PDF). Плотность в одной точке - это не вероятность. Вероятность получается интегрированием плотности по интервалу.

```
PDF: f(x)

P(a <= X <= b) = интеграл f(x) от a до b

f(x) может быть больше 1 (плотность, не вероятность)
интеграл f(x) dx от -inf до +inf = 1
```

Различие важно в ML. Выходы классификаторов - это PMF (дискретный выбор). Скрытые пространства VAE используют PDF (непрерывные).

### Распространенные распределения

**Бернулли:** один опыт, два исхода. Моделирует бинарную классификацию.

```
P(X = 1) = p
P(X = 0) = 1 - p
Среднее = p,  Дисперсия = p(1-p)
```

**Категориальное:** один опыт, k исходов. Моделирует многоклассовую классификацию (выход softmax).

```
P(X = i) = p_i,  где сумма p_i = 1
Пример: P(кот) = 0.7,  P(собака) = 0.2,  P(птица) = 0.1
```

**Равномерное:** все исходы равновероятны. Используется для случайной инициализации.

```
Дискретное: P(X = k) = 1/n для k в {1, ..., n}
Непрерывное: f(x) = 1/(b-a) для x в [a, b]
```

**Нормальное (Гауссово):** колоколообразная кривая. Параметризуется средним (mu) и дисперсией (sigma^2).

```
f(x) = (1 / sqrt(2*pi*sigma^2)) * exp(-(x - mu)^2 / (2*sigma^2))

Стандартное нормальное: mu = 0, sigma = 1
  68% данных в пределах 1 sigma
  95% в пределах 2 sigma
  99.7% в пределах 3 sigma
```

**Пуассона:** количество редких событий в фиксированном интервале. Моделирует интенсивности событий.

```
P(X = k) = (lambda^k * e^(-lambda)) / k!
Среднее = lambda,  Дисперсия = lambda
```

### Математическое ожидание и дисперсия

Математическое ожидание - это взвешенное среднее исход.

```
Дискретно:   E[X] = сумма x_i * P(X = x_i)
Непрерывно:  E[X] = интеграл x * f(x) dx
```

Дисперсия измеряет разброс вокруг среднего.

```
Var(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2
Стандартное отклонение = sqrt(Var(X))
```

В ML ожидание появляется как функция потерь (средняя ошибка по распределению данных). Дисперсия говорит о стабильности модели. Высокая дисперсия градиентов означает шумное обучение.

### Совместные и маргинальные распределения

Совместное распределение P(X, Y) описывает две случайные величины вместе.

Пример совместного PMF (X = погода, Y = зонтик):

| | Y=0 (без зонта) | Y=1 (с зонтом) | Маргинальное P(X) |
|---|---|---|---|
| X=0 (солнце) | 0.40 | 0.10 | P(X=0) = 0.50 |
| X=1 (дождь) | 0.05 | 0.45 | P(X=1) = 0.50 |
| **Маргинальное P(Y)** | P(Y=0) = 0.45 | P(Y=1) = 0.55 | 1.00 |

Маргинальное распределение получается суммированием по другой переменной:

```
P(X = x) = сумма по всем y от P(X = x, Y = y)
```

Суммы по строкам и столбцам в таблице выше - это маргиналы.

### Почему нормальное распределение повсюду

Центральная предельная теорема: сумма (или среднее) многих независимых случайных величин сходится к нормальному распределению, независимо от исходного распределения.

```
Бросок 1 кубика:  равномерное распределение (плоское)
Среднее 2 кубиков:  треугольное (пиковое)
Среднее 30 кубиков: почти идеальная колоколообразная кривая

Это работает для ЛЮБОГО начального распределения.
```

Вот почему:
- Ошибки измерений примерно нормальны (много независимых маленьких источников)
- Инициализация весов в нейросетях использует нормальные распределения
- Шум градиентов в SGD примерно нормален (сумма многих выборочных градиентов)
- Нормальное распределение - распределение с максимальной энтропией для заданного среднего и дисперсии

### Логарифмические вероятности

Сырые вероятности вызывают численные проблемы. Умножение многих маленьких вероятностей вместе быстро приводит к underflow в ноль.

```
P(предложение) = P(слово1) * P(слово2) * ... * P(слово_n)
               = 0.01 * 0.003 * 0.02 * ...
               -> 0.0 (underflow после ~30 членов)
```

Логарифмические вероятности решают это. Умножения становятся сложениями.

```
log P(предложение) = log P(слово1) + log P(слово2) + ... + log P(слово_n)
                   = -4.6 + -5.8 + -3.9 + ...
                   -> конечное число (нет underflow)
```

Правила:
- log(a * b) = log(a) + log(b)
- логарифмические вероятности всегда <= 0 (так как 0 < P <= 1)
- более отрицательное = менее вероятное
- cross-entropy loss - это отрицательный логарифм вероятности правильного класса

### Softmax как вероятностное распределение

Нейросети выдают сырые оценки (logits). Softmax переводит их в валидное вероятностное распределение.

```
softmax(z_i) = exp(z_i) / sum(exp(z_j) для всех j)

Свойства:
  - Все выходы в (0, 1)
  - Все выходы суммируются в 1
  - Сохраняет относительный порядок входов
  - exp() усиливает различия между logits
```

Трюк softmax: вычесть максимум logit перед экспонентой, чтобы избежать переполнения.

```
z = [100, 101, 102]
exp(102) = переполнение

z_shifted = z - max(z) = [-2, -1, 0]
exp(0) = 1  (безопасно)

Тот же результат, нет переполнения.
```

Log-softmax объединяет softmax и log для численной стабильности. PyTorch использует это внутри для cross-entropy loss.

### Семплирование

Семплирование означает вытягивание случайных значений из распределения. В ML:
- Dropout случайно обнуляет некоторые нейроны
- Data augmentation семплирует случайные трансформации
- Языковые модели семплируют следующий токен из предсказанного распределения
- Диффузионные модели семплируют шум и постепенно его удаляют

Семплирование из произвольных распределений требует техник вроде inverse transform sampling, rejection sampling или reparameterization trick (используется в VAE).

## Практика

### Шаг 1: Основы вероятности

```python
import math
import random

def factorial(n):
    result = 1
    for i in range(2, n + 1):
        result *= i
    return result

def combinations(n, k):
    return factorial(n) // (factorial(k) * factorial(n - k))

def conditional_probability(p_a_and_b, p_b):
    return p_a_and_b / p_b

p_king_given_face = conditional_probability(4/52, 12/52)
print(f"P(Король | Фигура) = {p_king_given_face:.4f}")
```

### Шаг 2: PMF и PDF с нуля

```python
def bernoulli_pmf(k, p):
    return p if k == 1 else (1 - p)

def categorical_pmf(k, probs):
    return probs[k]

def poisson_pmf(k, lam):
    return (lam ** k) * math.exp(-lam) / factorial(k)

def uniform_pdf(x, a, b):
    if a <= x <= b:
        return 1.0 / (b - a)
    return 0.0

def normal_pdf(x, mu, sigma):
    coeff = 1.0 / (sigma * math.sqrt(2 * math.pi))
    exponent = -0.5 * ((x - mu) / sigma) ** 2
    return coeff * math.exp(exponent)
```

### Шаг 3: Математическое ожидание и дисперсия

```python
def expected_value(values, probabilities):
    return sum(v * p for v, p in zip(values, probabilities))

def variance(values, probabilities):
    mu = expected_value(values, probabilities)
    return sum(p * (v - mu) ** 2 for v, p in zip(values, probabilities))

die_values = [1, 2, 3, 4, 5, 6]
die_probs = [1/6] * 6
mu = expected_value(die_values, die_probs)
var = variance(die_values, die_probs)
print(f"Кубик: E[X] = {mu:.4f}, Var(X) = {var:.4f}, SD = {var**0.5:.4f}")
```

### Шаг 4: Семплирование из распределений

```python
def sample_bernoulli(p, n=1):
    return [1 if random.random() < p else 0 for _ in range(n)]

def sample_categorical(probs, n=1):
    cumulative = []
    total = 0
    for p in probs:
        total += p
        cumulative.append(total)
    samples = []
    for _ in range(n):
        r = random.random()
        for i, c in enumerate(cumulative):
            if r <= c:
                samples.append(i)
                break
    return samples

def sample_normal_box_muller(mu, sigma, n=1):
    samples = []
    for _ in range(n):
        u1 = random.random()
        u2 = random.random()
        z = math.sqrt(-2 * math.log(u1)) * math.cos(2 * math.pi * u2)
        samples.append(mu + sigma * z)
    return samples
```

### Шаг 5: Softmax и логарифмические вероятности

```python
def softmax(logits):
    max_logit = max(logits)
    shifted = [z - max_logit for z in logits]
    exps = [math.exp(z) for z in shifted]
    total = sum(exps)
    return [e / total for e in exps]

def log_softmax(logits):
    max_logit = max(logits)
    shifted = [z - max_logit for z in logits]
    log_sum_exp = max_logit + math.log(sum(math.exp(z) for z in shifted))
    return [z - log_sum_exp for z in logits]

def cross_entropy_loss(logits, target_index):
    log_probs = log_softmax(logits)
    return -log_probs[target_index]
```

### Шаг 6: Демонстрация центральной предельной теоремы

```python
def demonstrate_clt(dist_fn, n_samples, n_averages):
    averages = []
    for _ in range(n_averages):
        samples = [dist_fn() for _ in range(n_samples)]
        averages.append(sum(samples) / len(samples))
    return averages
```

## Ключевые термины

| Термин | Что обычно говорят | Что это на самом деле |
|--------|---------------------|-----------------------|
| Пространство элементарных событий | "Все возможности" | Множество S всех возможных исходов эксперимента |
| PMF | "Функция вероятности" | Функция, которая дает точную вероятность каждого дискретного исхода, суммируясь в 1 |
| PDF | "Вероятностная кривая" | Функция плотности для непрерывных величин. Интегрируйте ее по интервалу, чтобы получить вероятность |
| Условная вероятность | "Вероятность при условии чего-то" | P(A\|B) = P(A и B) / P(B). Основа байесовского мышления и теоремы Байеса |
| Независимость | "Они не влияют друг на друга" | P(A и B) = P(A) * P(B). Знание одного события ничего не говорит о другом |
| Математическое ожидание | "Среднее" | Вероятностно-взвешенная сумма всех исходов. Функция потерь - это математическое ожидание |
| Дисперсия | "Насколько разбросано" | Ожидаемое квадратичное отклонение от среднего. Высокая дисперсия = шумные, нестабильные оценки |
| Нормальное распределение | "Колоколообразная кривая" | f(x) = (1/sqrt(2*pi*sigma^2)) * exp(-(x-mu)^2/(2*sigma^2)). Везде благодаря ЦПТ |
| Центральная предельная теорема | "Средние становятся нормальными" | Среднее многих независимых выборок сходится к нормальному распределению независимо от источника |
| Совместное распределение | "Две переменные вместе" | P(X, Y) описывает вероятность каждой комбинации исходов X и Y |
| Маргинальное распределение | "Суммируем по другой переменной" | P(X) = sum_y P(X, Y). Восстанавливает распределение одной переменной из совместного |
| Логарифмическая вероятность | "Логарифм вероятности" | log P(x). Превращает произведения в суммы, предотвращая численный underflow в длинных последовательностях |
| Softmax | "Преобразуем оценки в вероятности" | softmax(z_i) = exp(z_i) / sum(exp(z_j)). Отображает вещественные logits в валидное вероятностное распределение |
| Cross-entropy | "Функция потерь" | -sum(p_true * log(p_predicted)). Измеряет, насколько различны два распределения. Меньше - лучше |
| Logits | "Сырые выходы модели" | Ненормализованные оценки до softmax. Названы в честь логистической функции |
| Семплирование | "Вытягивание случайных значений" | Генерирование значений согласно вероятностному распределению. Как модели генерируют выходы |
